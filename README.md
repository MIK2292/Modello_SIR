# SIR pandemic model
This was a homework done for the course "Physics of the Complex Systems" by Vincenzo Carbone in the Academic year 2024/25 at University of Calabria.  

### Introduction
During the course of Complex Systems, we studied the SIR model for the evolution of a pandemic in a population. In this work I propose the study of this same model, considering an interaction that brings this model closer to reality. First we will see the differences between the mathematical definition of the Sir model and the improved SIR model, then we will find the equilibrium point for the improved model. We will then find the a formula to find R, (the number of removed) which is also independent on the mumber of susceptive and infected. The main part of this work is simulating and study the system using different values of the parameters and different initial conditions.
